# WorkflowPro - Android & PHP Integration

A sophisticated landing page showcasing the architecture of a native Android application integrated with a PHP/MySQL backend.

## 🛠 Tech Stack
- **Mobile:** Kotlin, Android Volley Library
- **Backend:** PHP 8.2, MySQL
- **Web:** HTML5, CSS3 (Dual-Theme Engine), JavaScript

## ✨ Key Features
- **Dynamic Theming:** Seamless transition between Light Mode and Neon Blue Dark Mode.
- **Architecture Showcase:** Visual representation of data flow between Mobile and SQL Database.
- **Responsive Design:** Optimized for all screen sizes using CSS Grid and Flexbox.

## 📱 About the Project
WorkflowPro is designed for enterprise-level employee management, focusing on real-time attendance tracking and payroll synchronization.
